Booting process of Windows
https://en.wikipedia.org/wiki/Booting_process_of_Windows

MS-DOS starts WIN.COM. In Windows 3.x, the WIN.COM starts KRNL286.EXE (STANDARD MODE) or KRNL386.EXE (386 ENHANCED MODE).


In all versions of Windows 9x except ME, it is also possible to load Windows by booting to a DOS prompt and typing "win".
There are some command line switches that can be used with the WIN command: with the /D switch, Windows boots to SAFE MODE, and with the /D:n switch, Windows boots to SAFE MODE with networking.
The latter switch only works properly with Windows 95.
In Windows 3.1, additional options are available, such as /3, which starts Windows in 386 ENHANCED MODE, and /S, which starts Windows in STANDARD MODE



Virtual DPMI host
	In computing, the DOS PROTECTED MODE Interface (DPMI) is a specification introduced in 1989 which allows a DOS program to run in protected mode, giving access to many features of the new PC processors of the time not available in REAL MODE. It was initially developed by Microsoft for Windows 3.0, although Microsoft later turned control of the specification over to an industry committee with open membership. Almost all modern DOS extenders are based on DPMI and allow DOS programs to address all memory available in the PC and to run in PROTECTED MODE.

#!/usr/bin/python3

"""
    get_categories.py

    MediaWiki API Demos
    Fetch and display categories associated with multiple Wikipedia pages.

    MIT License
"""

import requests

def get_categories(page_titles):
    """
    Fetch and print categories associated with given Wikipedia page titles.

    Args:
        page_titles (list): List of Wikipedia page titles.
    """
    S = requests.Session()
    URL = "https://en.wikipedia.org/w/api.php"

    for page_title in page_titles:
        PARAMS = {
            "action": "query",
            "format": "json",
            "prop": "categories",
            "cllimit": "500",
            "titles": page_title
        }

        try:
            R = S.get(url=URL, params=PARAMS)
            R.raise_for_status()
            DATA = R.json()

            PAGES = DATA.get("query", {}).get("pages", {})

            header = f"Categories for page '{page_title}':"
            print(header)
            print("-" * len(header))  # Underline with dashes

            for k, v in PAGES.items():
                if "categories" in v:
                    for cat in v['categories']:
                        print(cat["title"])
                else:
                    print("No categories found.")
            print("\n")  # Separate outputs for different pages with a new line

        except requests.exceptions.RequestException as e:
            print(f"An error occurred while fetching categories for '{page_title}': {e}")

if __name__ == "__main__":
    # Hardcoded list of page titles
    page_titles = ["CPU_modes", "Python_(programming_language)", "Artificial_intelligence"]
    get_categories(page_titles)
